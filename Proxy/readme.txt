Other examples of Proxy classes may be:

WCF Service Proxy classes : Encapsulates details (path, protocol and message serialization)of  how to invoke the service, and implements the same interface that the Real service does.
- Check the image in the project, it illustrates the usage of proxy class in WCF

Entity Framework proxy classes : Encapsulates CRUD operations, change tracking (created, deleted, changed), log, adds extra logic to the Entity classes.
